4. The app will switch themes based on time

**When to use:**
- You prefer dark mode at night
- You prefer light mode during the day
- You want automatic switching without thinking about it

## ğŸ’¡ Tips & Tricks

### Tip 1: Quick Access
The theme toggle is always visible in the navigation bar on every page. You never need to go to Settings to switch themes manually.

### Tip 2: Automatic Modes
Only one automatic mode can be active at a time:
- If you enable "Auto Dark Mode", "Time-Based Mode" turns off
- If you enable "Time-Based Mode", "Auto Dark Mode" turns off
- If you manually toggle the theme, both automatic modes turn off

### Tip 3: Battery Saving
Dark mode can save battery on OLED/AMOLED screens (most modern smartphones). Enable it to extend battery life!

### Tip 4: Eye Comfort
Dark mode reduces eye strain in low-light environments. Use it at night or in dark rooms for better comfort.

### Tip 5: Accessibility
Both themes maintain high contrast for readability. Choose whichever is more comfortable for your eyes.

## ğŸ”§ Troubleshooting

### Problem: Theme doesn't switch when I click the button
**Solution:**
- Refresh the page
- Clear browser cache
- Check if JavaScript is enabled

### Problem: Theme doesn't persist after closing browser
**Solution:**
- Check if cookies/localStorage are enabled
- Don't use private/incognito mode (it clears data)
- Check browser settings for data persistence

### Problem: Auto Dark Mode doesn't follow my device
**Solution:**
- Make sure "Auto Dark Mode" is enabled in Settings
- Check your device's theme setting
- Refresh the page after changing device theme

### Problem: Time-Based Mode doesn't switch at 6 PM/AM
**Solution:**
- Make sure "Time-Based Mode" is enabled in Settings
- Check your device's time and timezone
- Wait a minute (it checks every 60 seconds)

### Problem: Dark mode looks weird on some pages
**Solution:**
- This shouldn't happen! All pages support dark mode
- If you see issues, please report them
- Try refreshing the page
